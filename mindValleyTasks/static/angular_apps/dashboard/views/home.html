CV Uploader


Upload a JSON with particular fields that is validated and then saved in the Database.
Custom REST APIs have been built using Python (coupled with Django framework).
A Data validation class has been written that validates the uploaded JSON.
Custom methods for data fetching have been used.



Tic Tac Toe


The classic Tic Tac Toe game! Only you can't win this one!
Minimax algorithm has been used to create game tree and select best possible move.
The Minimax algorithm however fails for a 4X4 or 5X5 board, in that case Alpha-Beta pruning algorithm should be used.



URL Shortner


Shortens a given URL. The shortened and the original URL(s) are saved in the server.
A re-directable URL is returned that sends user to the original URL.



Miscellaneous Notes


  • I have user a Digital Ocean server that was spawned and set up from scratch. A single Django project has been used for all the tasks.

  • Installed Python2.7 with other needed dependencies such as dev tools, virtualenv, pip etc.

  • nGinx was used as Reverse proxy cum static file server. uWSGI is used as application server. Loggers have been enabled and monitored from reverse proxy to application server.

  • Git-flow is used for version control.

  • Angular is used for frontend, a SPA architecture has been implemented. Proper modules for services, controllers and utilities have been set up.

  • For server security, password based authentication has been disabled and SSH key based auth has been enabled. Application is running as a different user than root and none of the services are running as root.

  • SQLite database has been used right now, but can migrate to Postgres if needed.

  • If time would have permitted, I would have worked on the following features:

    Ansible deployment as compared to the simple bash script based deployment. Ansible deployment would allow remote deployments with automation for requirement installation, database migrations, static file settings etc.

    Implement the Alpha-beta prunning algorithm for NXN board.

    Write tests for my code, preferably shift to test driven development.

    Work on building a better UI in the frontend.
